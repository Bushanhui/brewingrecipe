Bloom = 원두량의 3배

First Pour = (원두량 X 비율 - Bloom) / 3 + Second Pour에서 반올림 및 빼기해서 모자란 물의 양

Second Pour = (원두량 X 비율 - Bloom) / 3 을 반올림하고 원두가 30g 이상인 경우 -20g, 30g 미만인 경우 -10g

Third Pour = (원두량 X 비율 - Bloom) / 3
